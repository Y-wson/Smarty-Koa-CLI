 ## Features

- 基于lerna的脚手架工具
- 内置路由、构建、部署、Lint 等，仅需一个 Smarty-Koa-CLI依赖即可上手开发。
- 多种Koa模板可供选择，快速生成项目


## Install

```
首先进行全局安装
npm i smarty-koa-cli -g

# OR

yarn global add smarty-koa-cli
```

## Quick Start
```
smarty-koa-cli init firstProject
```
